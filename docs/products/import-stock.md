# Import Stock from Shopify

Pull inventory levels from Shopify into Odoo.

## Manual Import

1. Navigate to **Shopify → Products → Import Stock**
2. Select your **Shopify Instance**
3. Select the **Location** to import from
4. Click **Import**

Odoo creates a stock adjustment to align quantities with Shopify.

:::warning
Importing stock from Shopify will create inventory adjustments in Odoo. Ensure you have the correct location mapping configured before running this operation.
:::
