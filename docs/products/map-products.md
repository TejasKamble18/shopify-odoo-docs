# Map Products

Link existing Odoo products to their Shopify equivalents without re-importing.

## When to Use Mapping

Use product mapping when:
- You already have products in Odoo and Shopify
- You want to link them without overwriting existing data
- You are migrating from another connector

## Mapping Steps

1. Navigate to **Shopify → Products → Product Mapping**
2. Click **New**
3. Select the **Odoo Product**
4. Enter the **Shopify Product ID**
5. Enter the **Shopify Variant ID** (if applicable)
6. Save

After mapping, the connector uses these IDs for all subsequent sync operations.
