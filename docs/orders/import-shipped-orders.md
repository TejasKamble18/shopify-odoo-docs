# Import Shipped Orders

Import already-fulfilled orders from Shopify — useful for historical imports or catch-up syncs.

## Steps

1. Navigate to **Shopify → Orders → Import Shipped Orders**
2. Select your **Shopify Instance**
3. Set a **Date Range**
4. Click **Import**

Shipped orders are imported as confirmed Sales Orders with validated delivery.
