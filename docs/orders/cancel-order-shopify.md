# Cancel Order in Shopify

Cancel a Shopify order from within Odoo.

## Steps

1. Open the Sales Order in Odoo
2. Click **Shopify → Cancel in Shopify**
3. Select the **Cancellation Reason**
4. Choose whether to **Restock Items**
5. Click **Confirm**

The order is cancelled in both Odoo and Shopify simultaneously.
