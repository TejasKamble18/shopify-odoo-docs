# Shopify–Odoo Connector Documentation

Welcome to the official documentation for the **SDLC Corp Shopify–Odoo Connector** — enterprise-grade real-time integration between Shopify and Odoo ERP.

## What is the Shopify–Odoo Connector?

The SDLC Corp Shopify–Odoo Connector is a production-ready module that synchronizes your Shopify store with Odoo ERP in real time. It handles products, orders, inventory, customers, payouts, and more — fully automated with zero custom development required.

## Key Capabilities

| Category | What's Synced |
|---|---|
| **Products** | Product listings, variants, prices, images |
| **Orders** | Unshipped, shipped, cancelled, refunded |
| **Inventory** | Stock levels, warehouse locations |
| **Customers** | Contact info, order history |
| **Financials** | Shopify payouts, net profit reports |
| **Webhooks** | Real-time event triggers |

## Supported Versions

- **Odoo**: v18,v19
- **Shopify**: All plans (Basic to Plus)
