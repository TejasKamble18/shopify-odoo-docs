# Shopify Net Profit Report

Analyze profitability of your Shopify sales directly in Odoo.

## Accessing the Report

Navigate to **Shopify → Reports → Net Profit Report**

## Report Breakdown

| Column | Description |
|---|---|
| **Revenue** | Total Shopify sales (ex. tax) |
| **COGS** | Cost of goods sold from Odoo products |
| **Gross Profit** | Revenue minus COGS |
| **Shopify Fees** | Transaction and processing fees |
| **Shipping Revenue** | Shipping charged to customers |
| **Shipping Cost** | Actual carrier costs |
| **Net Profit** | Final profit after all deductions |

## Requirements

- Products must have **Cost Price** set in Odoo
- Shopify Payments payout data must be imported for fee data
