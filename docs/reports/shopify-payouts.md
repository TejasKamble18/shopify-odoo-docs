# Shopify Payouts

Import Shopify Payments payout data into Odoo for reconciliation.

## Importing Payouts

1. Navigate to **Shopify → Reports → Shopify Payouts**
2. Select your **Shopify Instance** (must use Shopify Payments)
3. Click **Import Payouts**

## What Gets Imported

| Data | Description |
|---|---|
| **Payout Amount** | Net amount transferred to your bank |
| **Payout Date** | Date of bank transfer |
| **Transactions** | Individual order transactions within payout |
| **Fees** | Shopify processing fees |
| **Adjustments** | Refunds and chargebacks |

## Reconciliation

Imported payouts appear as bank statement lines in the configured Shopify Payments journal, ready for reconciliation in Odoo Accounting.
