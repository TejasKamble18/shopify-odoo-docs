# Shopify Sales Report

View consolidated sales data from your Shopify stores directly in Odoo.

## Accessing the Report

Navigate to **Shopify → Reports → Sales Report**

## Filters

| Filter | Description |
|---|---|
| **Instance** | Filter by Shopify store |
| **Date Range** | Set period (daily, weekly, monthly) |
| **Order Status** | Paid, Pending, Cancelled |

## Report Columns

- Order number and date
- Customer name
- Line items summary
- Subtotal, Tax, Discount, Total
- Payment method
- Fulfillment status
