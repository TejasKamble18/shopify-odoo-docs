Installation

Install the Shopify–Odoo Connector module and activate it in your Odoo environment.

📂 Getting Started v13–v18 ~5 min
odoo.yoursite.com/odoo/apps
📸 Odoo Apps menu → Update Apps List → Search "Shopify Connector" → Install

Download from the Odoo App Store or obtain via SDLC Corp's Odoo Integration Services.

Step-by-Step

1
Download the Module
Get the .zip from the Odoo App Store or directly from SDLC Corp.
2
Place on Addons Path
Extract and copy to your custom addons folder (e.g. /opt/odoo/custom_addons/). Ensure that path is listed in odoo.conf.
3
Restart Odoo
sudo systemctl restart odoo
4
Update Apps List
Log in as Administrator → Apps → Update Apps List.
5
Install the Connector
Search "Shopify Odoo Connector" and click Install.
✅
Done! The Shopify menu now appears in Odoo. Next, generate your API credentials.

odoo.conf Example

INI[options]
addons_path = /usr/lib/python3/dist-packages/odoo/addons,/opt/odoo/custom_addons
db_name     = your_database
http_port   = 8069
⚠️
Always restart Odoo and run Update Apps List after placing a new module on the addons path — without this step the module won't appear.

Generate API Key & Secret Key

Create a Custom App in Shopify Admin to produce the API credentials needed for the Odoo connection.

📂 Setup Connector Shopify Admin
admin.shopify.com/store/YOUR_STORE/settings/apps/development
📸 Shopify Admin → Settings → Apps and sales channels → Develop apps → Create an app
1
Open Shopify Admin
Settings → Apps and sales channels → Develop apps. Click Allow custom app development if prompted.
2
Create a Custom App
Click Create an app. Name it (e.g. SDLC Odoo Connector). Click Create app.
3
Configure API Scopes
Open Configuration → Admin API integration → Configure. Enable all required read/write scopes for products, orders, customers, inventory, fulfillments and payouts.
4
Install & Copy Credentials
Click Install app → go to API credentials tab. Copy your API key, API secret key and Admin API access token to a password manager immediately.
⚠️
One-time token — The Admin API access token is shown only once. If lost you must reinstall the Shopify app to generate a new one.

Configure Shopify Instance in Odoo

Link your Shopify store to Odoo using the API credentials generated in the previous step.

📂 Setup Connector Required Step
odoo.yoursite.com/odoo/shopify/instances/new
📸 Shopify → Configuration → Shopify Instances → Create → fill in API credentials → Test Connection
1
Navigate to Instances
Shopify → Configuration → Shopify Instances
2
Create New Instance
Click New. Enter the store name. You can create one instance per Shopify store.
3
Enter API Credentials
Shopify URL: mystore.myshopify.com
API Key, API Secret, Access Token: from your Shopify Custom App.
4
Test Connection
Click Test Connection. A green success banner confirms the link is working.
5
Save
Click Save. The instance is active and ready for sync operations.
💡
Multiple Stores — Create a separate instance per Shopify store. All share the same Odoo backend. SDLC Corp can help configure multi-store environments.
