## What is Project Cost Management?

The SDLC Corp Project Cost Management module gives you real-time visibility into project costs, budgets and profitability — all inside Odoo. Track expenses, forecast costs, allocate resources and generate detailed financial reports without leaving your ERP.

## Key Capabilities

| Category | What's Included |
|---|---|
| **Budgets** | Project budget creation, allocation, variance tracking |
| **Cost Tracking** | Real-time expense capture, cost code mapping |
| **Forecasting** | Estimated vs actual cost analysis, EAC calculations |
| **Resources** | Labour cost allocation, resource utilisation |
| **Reports** | Profitability reports, cost dashboards, Excel export |

## Supported Versions

- **Odoo**: v18, v19
